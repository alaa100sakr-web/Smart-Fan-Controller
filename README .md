# Smart Fan Controller

## Description
This project controls a fan based on temperature using a microcontroller.

## Components
- Microcontroller (AVR/Arduino)
- DHT11 sensor
- LCD display
- Motor driver

## Files
- main.c → main program
- LCD.c/h → LCD driver
- DHT11.c/h → sensor code
- project.hex → compiled file
